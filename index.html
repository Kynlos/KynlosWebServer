Getting Started

  1. Place your web files in the same directory as the server script.
  2. Ensure you have PHP installed and accessible from the command line.
  3. Run the server using the command: python main.py
  4. Open your browser and navigate to http://localhost:8080 (or the port you configured).

File Upload

You can upload files to the server by sending a POST request to /upload. The uploaded files will be saved in the current directory.

Rate Limiting

The server implements rate limiting to prevent abuse. By default, the server allows up to 10 requests per minute per client. This can be configured in the config.json file.

HTTPS Support

The server supports HTTPS with automatic self-signed certificate generation. To enable HTTPS, set "use_https": true in the config.json file. The server will generate a self-signed certificate and key if they do not already exist.

Configuration

The server reads configuration from a config.json file. Here is an example configuration:

{
    "port": 8080,
    "entry_point": "index.html",
    "use_https": false,
    "certfile": "cert.pem",
    "keyfile": "key.pem",
    "rate_limit": 10,
    "log_file": "server.log",
    "whitelist": [],
    "blacklist": [],
    "cert_config": {
        "C": "US",
        "ST": "California",
        "L": "San Francisco",
        "O": "My Company",
        "OU": "My Organization",
        "CN": "localhost",
        "serial_number": 1000,
        "valid_days": 3650
    }
}
